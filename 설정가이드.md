# 🍽️ 사내 맛집 지도 — 설정 가이드

## 최종 스택 (API 키 0개, 비용 $0)

| 항목 | 선택 | 비용 |
|------|------|------|
| 지도 | Leaflet + OpenStreetMap | 완전 무료 |
| DB | 노션 | 무료 (기존 사용 중) |
| Geocoding | Nominatim | 완전 무료 |
| 호스팅 | Netlify | 무료 플랜 |

> **API 키는 노션 토큰 2개뿐입니다.** 이 키들은 Netlify 환경변수에만 보관되고 HTML에는 전혀 없어요.

---

## 파일 구조

```
저장소/
  ├── foodmap.html                  ← 메인 앱 (API 키 없음)
  ├── netlify.toml                  ← Netlify 라우팅
  └── netlify/
      └── functions/
          └── api.js                ← 서버리스 함수 (노션 토큰 보관)
```

---

## STEP 1 — 노션 DB 생성

1. 노션 → 새 페이지 → 데이터베이스 → 전체 페이지 생성
2. 아래 컬럼을 **정확히 이 이름으로** 추가

| 컬럼명      | 타입    |
|------------|---------|
| Name       | 제목    |
| Addr       | 텍스트  |
| AddrDetail | 텍스트  |
| Cat        | 선택 (한식, 중식, 일식, 양식, 카페, 기타) |
| Stars      | 숫자    |
| Review     | 텍스트  |
| Nick       | 텍스트  |
| Lat        | 숫자    |
| Lng        | 숫자    |
| CreatedAt  | 텍스트  |
| IsSeeded   | 체크박스|

3. URL에서 Database ID 복사 (32자리 hex)

---

## STEP 2 — 노션 Integration 발급

1. https://www.notion.so/my-integrations → 새 통합 생성
2. Internal Integration Token 복사 (`secret_...`)
3. 노션 DB 페이지 `...` → **연결 추가** → 방금 만든 통합 선택

---

## STEP 3 — GitHub에 올리기

```bash
git init
git add foodmap.html netlify.toml netlify/functions/api.js
git commit -m "사내 맛집 지도"
git push origin main
```

---

## STEP 4 — Netlify 배포 + 환경변수

1. https://netlify.com → Add new site → Import from GitHub
2. 저장소 선택 → Deploy
3. **Site → Configuration → Environment variables** 에 3개 추가

| 변수명          | 값                              |
|----------------|----------------------------------|
| `NOTION_TOKEN`  | secret_xxxx                     |
| `NOTION_DB_ID`  | 32자리 Database ID              |
| `ALLOWED_ORIGIN`| https://yourteam.netlify.app    |

4. Deploys → **Trigger deploy** 로 재배포

---

## STEP 5 — 직원들에게 URL 공유

```
https://yourteam.netlify.app/foodmap.html
```

팀 이름으로 변경: Site configuration → Site details → **Change site name**

---

## 보안 체크리스트

- [ ] `foodmap.html` 에 노션 토큰 없는지 확인
- [ ] Netlify 환경변수에만 `NOTION_TOKEN`, `NOTION_DB_ID` 입력됐는지 확인
- [ ] `ALLOWED_ORIGIN` 이 팀 Netlify URL로 설정됐는지 확인
